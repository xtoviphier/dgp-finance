# DGP Finance — 8-Day AI Challenge Submission

> Accurate, offline financial statement generator for students & small businesses

## 🎯 Problem Solved
Manual financial reporting is time-consuming and error-prone for non-experts. DGP Finance automates Balance Sheet, Income Statement, and Cash Flow generation with validation and clear UI.

## ✨ Key Features
- ✅ **100% Offline**: No APIs, no internet required
- ✅ **9-Language UI**: English, Arabic, French, Spanish, Portuguese, Russian, German, Swahili, Mandarin
- ✅ **Input Validation**: Prevents accounting equation errors
- ✅ **Opt-In Forecasting**: OLS regression projection (separate tab, clear disclaimer)
- ✅ **Local AI Glossary**: Plain-English term hints via keyword matching
- ✅ **Streamlit UI**: Clean, judge-friendly, mobile-responsive

## 🛠️ Tech Stack
- Python 3.10+
- Streamlit (UI)
- pandas/numpy (calculations)
- scipy (OLS regression)
- Git (version control)
